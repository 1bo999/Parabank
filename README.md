  <em>Automated UI Testing Framework built with Java, Selenium, TestNG and Maven</em>
  for validating user journeys on the
  <a href="https://parabank.parasoft.com/parabank/index.htm" target="_blank"><strong>Parabank Web Bank</strong></a>
</p>

<hr>

<h2 style="margin-left: 20px">Overview</h2>

<p style="margin-left: 20px">
  This project is a <strong>Test Automation Framework</strong> created to ensure the functional stability and reliability of the Parabank website.
  <br>
  It follows the <strong>Page Object Model (POM)</strong> and emphasizes readability, reusability and easy maintenance.
</p>

<hr>

<h2 style="margin-left: 20px">Key Features</h2>

<div style="margin-left: 20px">
<p> <strong>POM</strong> for Clean separation of test logic and UI elements</p>
<p> <strong>Reusable Methods</strong> used as consistent and efficient scripting for Common utilities</p>
<p> <strong>Parameterized Testing</strong> for Dynamic data-driven test execution</p>
<p> <strong>Cross-Browser Support</strong> Chrome, Firefox, Edge</p>
<p> <strong>Configurable Environment</strong> `configuration.properties` support</p>
<p> <strong>Comprehensive Reporting</strong> TestNG HTML reports with logs</p>

</div>
